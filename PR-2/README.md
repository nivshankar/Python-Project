# Pattern Generator and Number Analyzer 🧮✨

## Project Overview
This Python project is a console-based application that provides two main features:

1. **Pattern Generation**
   - Generates a right-angled triangle star (`*`) pattern.
   - User can specify the number of rows.
   - Input validation ensures rows are between 2 and 15.

2. **Number Analyzer**
   - Checks whether numbers in a given range are odd or even.
   - Calculates the sum of numbers in the range.
   - Includes validation for incorrect ranges.

The program demonstrates:
- Loops
- Conditional statements
- Match-case structure
- Input validation

---

## Features 🚀
- Interactive menu-driven program
