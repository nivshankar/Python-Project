
# 🎓 Student Data Organiser

A simple Python console-based project to manage student records using dictionaries, tuples, and sets.  
This project allows users to add, update, display, and delete student information through a menu-driven interface.

---

## 🚀 Features

- Add new student records
- Prevent duplicate Student IDs
- Display all student details
- Update student information
  - Name
  - Subjects
  - Age
- Delete student records
- Display all unique subjects offered
- Menu-driven console interface

---

## 🛠️ Technologies Used

- Python 3.13
- Dictionaries
- Tuples
- Sets
- Match-Case Statements

## ▶️ How to Run

1. Install Python 3.x

```bash

```
